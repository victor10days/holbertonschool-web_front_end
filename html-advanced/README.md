Advanced HTML — Techium
Description
This project is part of the Holberton School Web Front End curriculum.
You will build the HTML structure of a fictional digital agency website called Techium, focusing strictly on semantic, accessible, standards-compliant HTML5 — no CSS, no JavaScript.
By the end, you’ll have:
A multi-page Techium website (homepage + internal pages)
A full HTML styleguide demonstrating core HTML elements
Clean, consistent structure ready to be styled in later projects
The website will look plain on purpose. The goal here is markup quality, hierarchy, and semantics — not design.
Learning Objectives
You should be able to explain, without external help:
Best practices & guidelines for writing HTML
How to create the skeleton of an HTML5 page
How and when to use semantic tags:
header, main, footer, nav, section, article, aside
When to use div vs span
Proper use and hierarchy of headings (h1–h6)
How to:
Create ordered, unordered, and definition lists
Embed and differentiate media types (SVG, GIF, PNG, JPG)
Structure tabular data with table, thead, tbody, th, tr, td
Embed video, audio, and external content (iframe)
How to correctly structure a full HTML page and ensure logical document flow
Requirements
A README.md at the root of the html_advanced directory is mandatory.
All HTML files must:
Use <!DOCTYPE html> and valid HTML5 structure
Use lowercase tags and attributes
Be as W3C-compliant as possible (using the W3C Validator), except where the project explicitly allows otherwise.
No CSS, no inline styling – structure only.
Techium is the brand name used across all pages.
Favicon and logo assets must be placed according to the tasks.

What You Build (Overview of Tasks)
Across tasks 0–40 you progressively:
Set up the HTML5 skeleton with doctype, html, head, and body.
Add meta tags, title, description, and favicons.
Introduce layout structure using header, main, and footer.
Add section, article, and aside to organize:
Hero
Services
Works
About Us
Latest news
Testimonials
Contact
Build a navigation bar and multi-page structure:
index.html, about.html, latest_news.html, contact.html
Internal anchor links and logo link to home.
Add headings hierarchies (h1–h3) and placeholder content.
Create a styleguide:
Headings
Paragraphs & text styles
Lists (ul, ol, dl)
Horizontal rules
Quotes & blockquotes
Addresses & inline semantics (e.g. mark, pre, code)
Table example
details / summary
Video, audio, and iframe embeds
Add logos, images, testimonials, social media links and SVG icons.
Use comments and structural wrappers (div, span) meaningfully to keep the code readable.
